my_script
=========

* before

```ruby
sudo yum install zsh
sudo yum install vundle
```

* install

1. git clone git://github.com/zlx/my_script.git

2. cd my_script

3. cp oh-my-zsh ~/.oh-my-zsh -r

4. cp script/.zshrc ~/.zshrc

5. cp script/.bash_alias ~/.bash_alias

6. cp script/.vimrc ~/.vimrc

7. vim && :BundleInstall

```
scripts
  ---> bin

alias
  ---> script/bash_alias
```
